# 超级牛冒险 🦞

死亡细胞风格的横版肉鸽闯关游戏，用 Phaser 3 开发。

## 操作

| 按键 | 动作 |
|------|:----:|
| ← → / A D | 左右移动 |
| ↑ / W / Space | 跳跃 |
| J | 攻击 |
| K / Shift | 翻滚（无敌） |

## 运行

浏览器直接打开 `index.html`，或者部署到 GitHub Pages 即可。
